ThinkPHP 6.0
===============

> 运行环境要求PHP7.1+。


## 安装

~~~
composer create-project topthink/think tp 6.0.*-dev
~~~

如果需要更新框架使用
~~~
composer update topthink/framework
~~~

## 参与开发

请参阅 [ThinkPHP 核心框架包](https://github.com/top-think/framework)。
